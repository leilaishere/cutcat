# 💕 可愛粉色互動網頁

專為特別的人設計的馬卡龍粉色調互動網頁，包含可愛的貓頭、小動物背景和甜蜜訊息。

---

## ✨ 特點

- 🎨 **馬卡龍粉色調** - 柔和夢幻的淡粉色漸變背景
- 🐱 **可愛貓頭** - 圓形貓頭作為封面主角
- 🎵 **背景音樂** - 點擊貓頭自動播放甜蜜音樂
- 🐾 **小動物環繞** - 8隻可愛的貓狗圍繞在貓頭周圍
- 😘 **親親特效** - 點擊後出現大量飛舞的親親表情
- 💝 **甜蜜訊息** - 粉色漸變訊息框配紫色邊框和蝴蝶裝飾
- 📱 **移動端優化** - 完美適配手機瀏覽
- ✨ **浮動動畫** - 小動物和裝飾都有輕柔的浮動效果

---

## 🎀 互動效果

### 封面頁：
- 貓頭在粉色背景中央
- 8隻小動物以圓形排列環繞貓頭
- 所有元素帶有輕柔浮動動畫
- 提示文字「點擊我！💕」

### 點擊貓頭後：
1. 🎵 背景音樂自動播放
2. 😘 50+ 親親表情從各處飛出
3. 💌 顯示訊息框：「帥哥，以後就讓我來保護你」
4. 🦋 訊息框邊緣有蝴蝶和星星裝飾
5. 🐾 背景小動物保持顯示
6. 💕 點擊畫面任意處繼續產生親親特效

---

## 🎨 設計細節

### 配色方案：
- **背景**：馬卡龍淡粉色漸變
- **邊框**：深紫羅蘭色（6px 粗邊框）
- **文字**：深粉紅色
- **裝飾**：藍色蝴蝶 🦋、星星 ✨、愛心 💕

### 裝飾元素：
- 訊息框四角：💕✨💖🌸（浮動）
- 邊框裝飾：🦋💫🌟✨💝（旋轉+飛舞）
- 背景愛心：💕💖💗💝💓（隨機分佈）

## 📦 本地運行

1. 克隆倉庫：
```bash
git clone https://github.com/leilaishere/thankyoucat.git
```

2. 打開文件：
```bash
cd thankyoucat
open cute-page.html
```

或者直接雙擊 `cute-page.html` 文件在瀏覽器中打開。

## 🛠️ 技術特色

- **純 HTML + CSS + JavaScript**（無需框架）
- **CSS 動畫**：浮動、旋轉、飛舞、淡入淡出
- **混合模式**：去除圖片白色背景
- **圓形裁剪**：`clip-path` 實現完美圓形
- **響應式設計**：移動端完美適配
- **音頻 API**：自動播放循環音樂

## 📝 自定義

你可以輕鬆修改：

### 圖片：
- 替換 `cat.jpg` 為你的圖片
- 替換 `图像 2-9.jpg` 為其他可愛圖片

### 文字：
```javascript
// 在 cute-page.html 中找到：
<div class="message-text">
    帥哥，以後就讓我來保護你
</div>
// 修改為你想要的文字
```

### 音樂：
- 替換 `my-favourite-day-with-you-427760.mp3` 為你的音樂文件

### 配色：
```css
/* 背景顏色 */
background: linear-gradient(135deg, #FFD6E0 0%, #FFC9D6 50%, #FFB8C8 100%);

/* 邊框顏色 */
border: 6px solid rgba(147, 112, 219, 0.95);

/* 文字顏色 */
color: #FF1493;
```
